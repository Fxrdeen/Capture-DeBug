Given an array of size n, find the minimum excluded element (MEX).
The MEX (minimum excluded) of a set is the smallest non-negative integer that does not belong to the set.

Input Format
The first line contains the integer n.
The second line contains n space seperated integers.

Constraints
1 <= n <= 1e5
1 <= a[i] <= 1e5

Output Format
Output a single integer. The minimum excluded element of the array.

Sample Input 0

5
1 2 4 5 6

Sample Output 0
3
